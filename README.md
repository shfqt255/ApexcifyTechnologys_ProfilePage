# Profile Manager – Flutter App

This Flutter application was developed as part of an internship assignment given by **Apexify Technologies**.  
It demonstrates adding, editing, and saving profile information locally, along with image selection functionality.

---

## Features

- Add **Name** , **Email**, **Bio**, and **Profile Image** (Camera & Gallery)
- Edit and update profile information
- Persist data using **Shared Preferences**
- Live validation for form fields
- Display selected image in the profile screen

---

## Tech Stack

- **Flutter** – UI framework  
- **Dart** – Programming language  
- **Provider** – State management  
- **Shared Preferences** – Local data storage  
- **Image Picker** – Camera and gallery access
